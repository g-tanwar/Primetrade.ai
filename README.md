# Primetrade.ai

Minimal Node.js Express backend starter with a clean, scalable structure.

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Create env file:

```bash
cp .env.example .env
```

3. Run in development:

```bash
npm run dev
```

Server runs on `http://localhost:5000` by default.

## Folder Structure

`src/routes` - endpoint definitions
`src/controllers` - request handling logic
`src/models` - data layer placeholders
`src/middleware` - reusable middleware
